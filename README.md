
# Pshop18

A Prototype E-commerce site using PHP and SQL database.

## Features

- Authorization (Registration & Login)
- Manage Account
- Products & Category Search
- Shopping Cart
- CRUD Operations
  
## Tech Stack

**Client:** HTML, CSS, JS, JQuery, Bootstrap & Materialize CSS

**Server:** PHP

**Database:** MySQL
